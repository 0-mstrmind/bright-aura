# ExpressJS app (Backend) For Bright Aura - Streetlight Monitoring System

- This ExpressJS backend app is developed for Hardware (Gateway -> Server) Data transfer
- Corrently developed for  Bright Aura - Streetlight Monitoring System (Hack The Winter Round - 1 Prototype)

- Used seperately from NextJS Web Dashbord app (Independent)

- Need correct environment variable values in ".env" file to work, Please check ".env.example" file for reference!
